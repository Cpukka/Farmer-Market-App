# Farmer Market Connect 🌱

A modern, full-featured farmer's marketplace connecting local farmers directly with consumers. Built with Next.js 14, TypeScript, and Tailwind CSS.

## Features

- 🛍️ **Marketplace** - Browse and purchase fresh local produce
- 👨‍🌾 **Farmer Profiles** - Connect directly with local farmers
- 🛒 **Shopping Cart** - Easy checkout process
- 🔐 **Authentication** - NextAuth.js with multiple providers
- 🌙 **Dark Mode** - Full dark/light theme support
- 📱 **Responsive Design** - Works on all devices
- 🗄️ **Database** - PostgreSQL with Prisma ORM
- 📊 **Dashboard** - Analytics and management for farmers
- ⭐ **Reviews & Ratings** - Customer feedback system

## Tech Stack

- **Frontend**: Next.js 14, TypeScript, Tailwind CSS 4
- **Backend**: Next.js API Routes, Prisma ORM
- **Database**: PostgreSQL
- **Authentication**: NextAuth.js
- **Styling**: Tailwind CSS, Radix UI Components
- **Icons**: Lucide React
